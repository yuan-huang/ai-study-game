# Gemini LangChain 测试项目

这是一个使用Google Gemini API和LangChain框架的测试项目。

## 安装

```bash
npm install
```

## 配置

项目使用以下环境变量：

- `GEMINI_API_KEY`: Google Gemini API密钥

## 运行

```bash
node index.js
```

## 功能

- 使用LangChain的提示模板
- 集成Google Gemini API
- 简单的错误处理
- 支持自定义问题输入

## 注意事项

确保你有有效的Google Gemini API密钥才能使用此项目。
