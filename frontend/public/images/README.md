## 注意事项
1. 所有图片建议使用PNG格式，保持透明背景
2. 图片命名使用英文小写，单词间用连字符"-"连接
3. 按钮类图片建议预留点击状态（normal/hover/pressed）
4. 建议使用矢量工具设计，以保证图片质量
